# Air Raid Alerts Time Series Analysis

Анализ и прогнозирование воздушных тревог в Украине с использованием методов анализа временных рядов.

## 📊 Описание проекта

Этот проект фокусируется на:
- Анализе исторических данных о воздушных тревогах
- Выявлении паттернов и трендов
- Построении моделей прогнозирования
- Визуализации распределения тревог по времени и регионам

## 📁 Структура проекта

```
air-raid-alerts-analysis/
├── docs/                  # Документация
│   ├── session-history.md # История сессий разработки
│   ├── project-spec.md    # Спецификация проекта
│   └── implementation-plan.md # План реализации
├── src/                   # Исходный код
├── data/                  # Данные (raw и processed)
├── notebooks/             # Jupyter ноутбуки для EDA и анализа
├── tests/                 # Тесты
├── requirements.txt       # Зависимости Python
└── README.md             # Этот файл
```

## 🚀 Начало работы

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Загрузить данные в папку `data/`
4. Запустить анализ

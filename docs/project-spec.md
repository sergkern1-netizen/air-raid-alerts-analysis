# Air Raid Alerts Time Series Analysis — Project Specification

## 1. Обзор проекта

**Название:** Time Series Analysis of Air Raid Alerts in Ukraine

**Цель:** Разработать Python-приложение для анализа и прогнозирования временных рядов воздушных тревог в Украине, включая выявление паттернов, статистический анализ и построение предиктивных моделей.

**Тип:** Data Science Pet-Project

**Язык реализации:** Python 3.9+

## 2. Функциональные требования

### 2.1 Загрузка и обработка данных
- Получение данных о воздушных тревогах из открытых источников (API или CSV)
- Парсинг и валидация данных
- Обработка пропусков и выбросов
- Агрегирование данных по временным периодам (час, день, неделя)
- Геоспециальная информация (по регионам, если доступно)

### 2.2 Исследовательский анализ (EDA)
- Статистическое описание данных (mean, std, quantiles)
- Анализ распределений
- Выявление временных паттернов (суточные, недельные, сезонные)
- Корреляционный анализ между регионами
- Выявление аномалий и выбросов

### 2.3 Анализ временных рядов
- Декомпозиция временного ряда (тренд, сезонность, остаток)
- Анализ автокорреляции (ACF, PACF)
- Тесты стационарности (ADF test)
- Анализ спектральной плотности

### 2.4 Прогностические модели
- ARIMA/SARIMA для одномерных рядов
- Prophet для сезонных данных (опционально)
- LSTM/RNN для глубокого обучения (опционально)
- Ensemble методы для комбинирования прогнозов
- Кросс-валидация и оценка метрик (MAE, RMSE, MAPE)

### 2.5 Визуализация и отчеты
- Графики временных рядов (основная серия, агрегированные данные)
- Сезонные и трендовые компоненты
- Прогнозы с доверительными интервалами
- Сравнение разных моделей
- Интерактивные визуализации (опционально)

## 3. Технический стек

- **numpy** — численные вычисления

- **plotly** — интерактивные графики

- **statsmodels** — анализ временных рядов (ARIMA, декомпозиция)

- **tensorflow/keras** — глубокое обучение (опционально)

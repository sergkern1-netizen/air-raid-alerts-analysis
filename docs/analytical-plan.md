# План Подробного Аналитического Отчёта

## Цель
Создать профессиональное аналитическое исследование воздушных тревог в Украине с статистическими доказательствами, регрессионным анализом и визуализациями.

## Структура детального отчёта

### ЧАСТЬ 1: ОПИСОВАЯ СТАТИСТИКА И БАЗОВЫЙ АНАЛИЗ

**1. Дескриптивная статистика**
- Среднее, медиана, стандартное отклонение, квартили
- Асимметрия и эксцесс (проверка нормальности распределения)
- Выявление выбросов (IQR метод, Z-score)

**2. Тестирование нормальности**
- Shapiro-Wilk тест
- Kolmogorov-Smirnov тест
- Интерпретация результатов

**3. Анализ пропусков и качества данных**
- Процент пропусков по регионам
- Консистентность между источниками
- Детектирование аномалий

### ЧАСТЬ 2: ВРЕМЕННЫЕ РЯДЫ И ТРЕНДЫ

**4. Декомпозиция временного ряда**
- Trend компонента (линейный, полиномиальный тренд)
- Seasonal компонента (периоды, амплитуда)
- Residual компонента (белый шум, автокорреляция)
- Additive vs multiplicative модели

**5. Анализ стационарности**
- ADF (Augmented Dickey-Fuller) тест
- KPSS тест
- Интерпретация для моделирования

**6. Автокорреляция и частичная автокорреляция**
- ACF графики (lag структура)
- PACF графики (определение AR/MA порядков)
- Значимые лаги (95% доверительный интервал)

**7. Сезонность анализ**
- Месячные паттерны (тесты значимости)
- Квартальные тренды
- День недели эффекты (если данные часовые)
- Сезонный индекс по месяцам

### ЧАСТЬ 3: РЕГИОНАЛЬНЫЙ АНАЛИЗ

**8. Кластеризация регионов**
- K-means на основе интенсивности
- Иерархическая кластеризация
- Дендрограмма и эллиптический анализ

**9. Регрессионный анализ по регионам**
- Y = интенсивность тревог
- X = временные переменные (тренд, время года)
- R², коэффициенты, p-values
- Таблица сравнения регионов

**10. Анализ лагов между регионами**
- Кросс-корреляция между регионами
- Ведущие и отстающие регионы
- Причинно-следственные связи (Granger causality)

**11. Региональные риск-профили**
- Волатильность по регионам
- Максимальная однодневная интенсивность
- Вероятность >X тревог в день (по регионам)

### ЧАСТЬ 4: ПРИЧИННО-СЛЕДСТВЕННЫЙ АНАЛИЗ

**12. Корреляционный анализ**
- Пирсон корреляция между регионами
- Спирмен ранговая корреляция
- Матрица корреляций с тепловой картой

**13. Грэнжер причинность**
- Гипотеза: регион A предсказывает регион B
- F-статистика и p-values
- Направленность влияния (A→B или B→A)

**14. Анализ причин эскалации**
- Сравнение периодов (2023 vs 2025)
- Статистический тест различий (t-test, Mann-Whitney U)
- Размер эффекта (Cohen's d, eta-squared)

### ЧАСТЬ 5: ПРОГНОСТИЧЕСКИЙ АНАЛИЗ

**15. Сравнение моделей (детальный)**
- Кросс-валидация (5-fold, time-series split)
- Доверительные интервалы прогнозов
- Анализ ошибок по периодам (лучше летом или зимой?)
- Sensitivity analysis на гиперпараметры

**16. Ensemble методы**
- Weighted ensemble (оптимальные веса)
- Stacking модели
- Bayesian model averaging

**17. Forecast accuracy metrics**
- MAPE, MAE, RMSE по месяцам
- Diebold-Mariano тест (статистическая значимость разницы)
- Theil's U-statistic (сравнение с наивным прогнозом)

### ЧАСТЬ 6: ВИЗУАЛИЗАЦИИ И ИНФОГРАФИКА

**18. Временные ряды графики**
- Daily alerts с moving average (7-дневная, 30-дневная)
- Seasonal subseries plot
- Year-on-year comparison

**19. Распределения и box plots**
- Гистограммы по месяцам, регионам
- Box plots по регионам (выявление выбросов)
- Violin plots (распределение формы)

**20. Регрессионные графики**
- Scatter plots с линией регрессии и доверительным интервалом
- Residual plots (нормальность, гомоскедастичность)
- Q-Q plot

**21. Тепловые карты**
- Интенсивность по регионам и месяцам (heatmap)
- Корреляционная матрица
- Temporal heatmap (день недели + месяц)

**22. Карты эскалации**
- Процент изменения 2023→2024→2025 по регионам
- Геоспециальная визуализация (если возможно)

### ЧАСТЬ 7: ВЫВОДЫ И РЕКОМЕНДАЦИИ

**23. Ключевые статистические находки**
- Список с p-values и effect sizes
- Что статистически значимо vs случайное

**24. Интерпретация причинности**
- Что может объяснить эскалацию 2025?
- Какие регионы синхронизированы?
- Есть ли запаздывающие эффекты?

**25. Ограничения анализа**
- Confounding факторы
- Смещение в данных
- Предположения нарушены?

**26. Рекомендации для политики**
- На основе доказательств
- Приоритеты по регионам
- Метрики мониторинга

## Технические деталі

### Инструменты
- Python: pandas, scipy, statsmodels, scikit-learn
- Графики: matplotlib, seaborn, plotly
- Статистика: R (если нужны специализированные тесты)

### Данные для анализа
```
01_daily_aggregates.csv      - основной временной ряд (1563 дня)
02_regional_summary.csv      - сводка по 25 регионам
03_regional_daily.csv        - 26,751 записей (регион + день)
10_quarterly_pattern.csv     - квартальные тренды
16_regional_trends.csv       - тренды регионов
18_recent_escalation.csv     - западная эскалация (6 месяцев)
validated_combined.csv       - сыре данные (418K записей)
```

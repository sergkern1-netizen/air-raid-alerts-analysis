## Проект

Time Series Analysis of Air Raid Alerts in Ukraine — Python pet-project для анализа и прогнозирования воздушных тревог.

## Ключевые файлы

- Спек проекта: `docs/project-spec.md`
- План реализации: `docs/implementation-plan.md`
- История сессий: `docs/session-history.md`
- Исходный код: `src/`
- Jupyter ноутбуки: `notebooks/`

## Session Logging

В начале каждой новой сессии:
1. Прочитай `docs/session-history.md`
2. Добавь запись о новой сессии в конец файла (дата, что делали, ключевые решения)

В конце сессии или при завершении крупного этапа — обнови `docs/session-history.md`.

## Требования к проекту

- Python 3.9+
- pandas, numpy для работы с данными
- scikit-learn для ML моделей
- matplotlib, seaborn для визуализации
- statsmodels для анализа временных рядов

## Правила разработки

1. Автономное выполнение задач без лишних запросов подтверждения
2. Расширение существующего кода вместо переписывания
3. Ясное документирование процесса в session-history.md
